## INTERACTIVE INPUT

Implement the interact() method to gather missing information:

```php
protected function interact(InputInterface $input, OutputInterface $output): void
{
    $helper = $this->getHelper('question');

    // Text question
    if (!$input->getArgument('name')) {
        $question = new Question('Please enter the name: ');
        $name = $helper->ask($input, $output, $question);
        $input->setArgument('name', $name);
    }

    // Confirmation question
    $question = new ConfirmationQuestion('Continue with this action? (y/n) ', false);
    if (!$helper->ask($input, $output, $question)) {
        throw new \RuntimeException('Operation cancelled by user');
    }

    // Choice question
    $question = new ChoiceQuestion(
        'Please select an option:',
        ['option1', 'option2', 'option3'],
        0
    );
    $choice = $helper->ask($input, $output, $question);
}
```

## QUESTION TYPES

1. **Simple Text Input**:
```php
$question = new Question('Enter value: ', 'default_value');
$value = $helper->ask($input, $output, $question);
```

2. **Hidden Input (for passwords)**:
```php
$question = new Question('Enter password: ');
$question->setHidden(true);
$password = $helper->ask($input, $output, $question);
```

3. **Validation**:
```php
$question = new Question('Enter email: ');
$question->setValidator(function ($answer) {
    if (!filter_var($answer, FILTER_VALIDATE_EMAIL)) {
        throw new \RuntimeException('Invalid email address');
    }
    return $answer;
});
$email = $helper->ask($input, $output, $question);
```

4. **Choice Question**:
```php
$question = new ChoiceQuestion(
    'Select environment:',
    ['development', 'staging', 'production'],
    0
);
$environment = $helper->ask($input, $output, $question);
```

5. **Confirmation**:
```php
$question = new ConfirmationQuestion('Are you sure? (y/n) ', false);
if ($helper->ask($input, $output, $question)) {
    // User confirmed
}
```
